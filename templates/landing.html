DXF File Requirements

Technical specifications and best practices
Technical Requirements
  • AutoCAD DXF format (R14 or newer)
  • Room boundaries as closed polylines
  • Start points marked as points/circles
  • 2D plan view drawings
  • Metric units (millimeters preferred)
  • Maximum file size: 50MB
Best Practices
  • Clean, organized layer structure
  • Consistent naming conventions
  • Accurate room dimensions
  • Properly closed room boundaries
  • Start points within room boundaries
  • Remove unnecessary entities
Supported Tile Sizes

Common sizes: 300x300mm, 600x600mm, 800x800mm, 300x600mm, and custom dimensions. The system automatically detects tile dimensions from start point markers.
